# API Setup Guide

## OpenWeatherMap API Configuration

### Step 1: Get an API Key
1. Visit [OpenWeatherMap](https://openweathermap.org/api)
2. Click "Sign Up" for a free account
3. Verify your email address
4. Log in to your account
5. Navigate to the "API Keys" tab in your dashboard

### Step 2: Choose API Plan
- **Free Tier**: 1,000 calls/day, 60 calls/minute
- Suitable for personal use and development
- No credit card required

### Step 3: Configure the Application
1. Open `apikey.txt` in a text editor
2. Replace the example key with your actual API key:
   ```txt
   api_key = 'your_actual_32_character_api_key_here'
